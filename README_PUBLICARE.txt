📄 INSTRUCȚIUNI PUBLICARE SWISSROHUB PE GITHUB

1️⃣ Deschide terminalul (sau Git Bash pe Windows) în acest folder.

2️⃣ Execută următoarele comenzi una câte una:

    git init
    git remote add origin https://github.com/IacobDoru/iacobdoru.github.io.git
    git branch -M main
    git add .
    git commit -m "Publicare site SwissRoHub"
    git push -u origin main

3️⃣ Gata! Intră apoi pe:
    https://iacobdoru.github.io

Site-ul tău va fi online în 1-2 minute.
